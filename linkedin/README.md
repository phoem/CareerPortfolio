# LinkedIn Profile Records

This directory contains the durable audit trail for changes made through the manual LinkedIn profile workflow.

- `CHANGELOG.md` is append-only and records attempted and successful external profile changes.
- Optional preserved proposals and completion reports may be stored under `reviews/<YYYY-MM-DD>/` when Jordan requests them.
- Credentials, cookies, session tokens, recovery information, private messages, and unnecessary browser captures must never be stored here.

See `docs/workflows/LINKEDIN_PROFILE_WORKFLOW.md` for the approval and publishing process.
