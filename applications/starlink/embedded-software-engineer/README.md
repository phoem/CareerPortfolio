# Starlink — Embedded Software Engineer

This directory contains the historical application package for a Starlink Embedded Software Engineer role.

The original requisition ID and exact job posting were not preserved, so the package uses the verified role name as its slug and leaves those fields unknown.

## Package contents

- `APPLICATION.json` — machine-readable package manifest.
- `JOB_SOURCE.md` — known provenance and missing posting details.
- ATS and recruiter-focused resume sources with generated DOCX/PDF artifacts.
- cover-letter source with generated DOCX/PDF artifacts.
- `DESIGN.json` — persistent artifact design selection.
- `validation/` — ATS status; targeted scoring remains pending until the exact posting is recovered.
