# AVR Smart Smoke/CO2 Detector

## Summary

A contract embedded-firmware project for battery-powered wireless smart smoke and CO2 detector prototypes.

## Confirmed Implementation

- embedded C;
- Atmel AVR microcontrollers;
- SPI, UART, and I2C interfaces;
- Arduino-related development;
- circuit design, soldering, and hardware troubleshooting;
- battery-powered wireless prototypes.
